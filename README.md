## üì¶ Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/manish-reddy7/manish-Reddy-Portfolio-.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd manish-Reddy-Portfolio-
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Start the development server:
    ```bash
    npm run dev
    ```

## üö¢ Deployment

This project is ready to be deployed on any static hosting service.

### Vercel (Recommended)
1.  Push your code to GitHub.
2.  Import your repository into Vercel.
3.  Vercel will automatically detect Vite and deploy your site.

### Netlify
1.  Push your code to GitHub.
2.  Import your repository into Netlify.
3.  Set the build command to `npm run build` and publish directory to `dist`.

### GitHub Pages
1.  Update `vite.config.ts` to set the base URL if needed.
2.  Run `npm run build`.
3.  Deploy the `dist` folder.
